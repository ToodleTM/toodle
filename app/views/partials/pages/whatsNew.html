New features

The interface buttons are now more "menu" like. It seems to be less messy (all buttons are now in one clean line that does not move around)

Regrouped functions look & feel.

"Real" new feature : The Cup Management

Cup Management allows you to create follow-up tournaments once a tournament you've run has ended (i.e: winners are known). The cup system is embedded into the app and does not require you to choose a particular tournament at the beginning.

The features :

  • Once a tournament has ended, you can choose to follow it up with another tournament (the most straightforward example would be to inject winners from group stages into a bracket).
    The button to create follow-up tournaments will appear on the right of the menu bar once a tournament has ended.
  • If you choose to do so, Toodle will ask you for details about the new tournament (name, start date, description ... as it would when creating a normal tournament)
    The form is like the standard one. Just input a title, select your engine, and you're good to go =)
  • Depending on the number of winners and the nature of the follow-up tournament, you'll be able to start right away if you do so desire, but you'll also have all the normal options (as rearranging players in the follow up tournament). If it's not possible to start right away, you'll still be taken to the admin page of the follow-up tournament and asked to fix what needs to be fixed before starting.
  • Once a follow-up tournament has started, the initial tournament is locked (cannot edit it in any way that'd change the winners anymore)
  • In the follow-up tournament page, there is a button that allows you to display the previous tournament (aka: parent tournament) and its parent's parent, and so on until you move all the way up to the initial tournament.
    Button to show parent tournaments is always shown at the bottom
    Once clicked, the parent tournament is displayed. If that parent tournament also has a parent, the button stays visible and allows you to display the parent's parent.
  • There is no fixed limit on how many tournaments can be chained this way
  • On each tournament in a cup, you have menus to help you navigate between tournaments.
    Menus allow to access nearest parent / son tournament. Similar menus are displayed in the public page (just not showing admin links).
  • This was made to be able to mix tournaments of any engines (existing, of course, but also & kinda more importantly, upcoming). We don't want any limitations put on this so that a complete tournament going through different phases will be possible as long as tournament engines individually support what you're shooting for.

As a consequence, the "export winners" button once a tournament has finished has disappeared (for now, at least).

Sept. 20th 2015 (v1.2.3)

Not much to say about this version, some tweaks and fixes for the most part, plus one interesting feature : embedding.

Embedding is a feature that allows you to link to a tournament bracket from another site using iframes. The page to include is different from admin / play pages in a way that only the core information is available (current tournament standings, read-only).

August 24th 2015(v1.2.1)

This release represents a major GUI overhaul. This release is meant to :

  • Simplify even more tournament creation
  • Add a few missing features and improvements

On the GUI overhaul side :

  • Creating a tournament immediately takes you to the essential part : registering the players and defining the type of tournament you want
  • Admin and "regular player" pages were changed so that all tournament fine tuning is hidden and the first thing you see is your bracket. If you want to access to all fine tuning settings, just click the main title and the menu will deploy.

On the features side :

This release has seen the addition of various features that should be useful :

  • Both "Single Elimination Bracket" and "Simple GSL Groups" engines now support :
    • Player forfeits
    • Continuous reporting
    • Pre-arrangement of players before tournament starts (by clicking the spots you want to swap)

Various noticeable changes

What disappeared

The "lock tournament" feature is no more, it might be back someday, you never know ;)

What appeared

No more clicking validation buttons all the time, each action auto-saves what you just did.

Adding multiple players one by one always refocuses the nickname input field (provided addition went well)

More in detail :

Registration screen

It now integrates not only the tournament name, but you can also specify the description or the start date here.

it takes you directly to the next step once you validate

Player registration / engine choice

This is a new step, it allows you to configure the essentials (register some players, choose the engine, get the tournament started)

The two more important decisions for your tournament

The middle option creates the tournament but does not start it right away

Pre-arrangement / players move around

The "let me configure players" option, when it is available, takes you to a page that show you the tournament as it is arranged by default, given the players registered and the order they were registered in, and lets you move players around by clicking on a slot and then on another.

Forfeits

The match reporting dialog now allows for forfeits to be reported, you can forfeit a player by clicking on the little "" logo in the field for his score. The engines will then count this as a win for the other player and save the scores for that match

Forfeit crosses the forfeit player name out and locks all inputs
Forfeits are displayed differently from victories so you can keep track at a glance

Continuous reporting

The reporting feature now allows to save a partial score. When a score is valid to report a victory (scores are different).

A checkbox pops up for the reporter to select if the score reported is final or not

Admin page

Most fine tuning tools are now hidden in a menu, so that the first thing you see in the admin page is the actual bracket when a tournament has started

Clicking on the tournament name, or the caret next to it, will deploy this menu, clicking again will hide it.
The second tab show player management features.
